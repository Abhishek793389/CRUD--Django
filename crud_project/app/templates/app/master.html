User Management System

Discription : The User Management System is a web application designed to provide a seamless experience for managing user data. Built using the Django framework, this project implements the fundamental Create, Read, Update, and Delete (CRUD) operations, allowing users to interact with the database effectively.

Key features :

  • User Registration : Users can create new accounts by providing their details through a registration form. The information is securely stored in a MySQL database, ensuring data integrity and user confidentiality.

  • User Profile : Each registered user has a unique profile that displays their information, such as full name, mobile number, date of birth, and address. Users can view their profiles at any time.

  • Data Manipulation : Users have the ability to :

    • Create : Add new user records via the registration form.
    • Read : View all existing user profiles and their details.
    • Update : Edit user information to keep records up to date, such as changing contact details or updating addresses.
    • Delete : Remove user records when they are no longer needed, ensuring the database is clean and relevant.

Conclusion : The User Management System serves as an excellent demonstration of my ability to develop web applications with robust data handling capabilities. It showcases my skills in Django, Python and database management. I am excited to continue enhancing this project with new features and improvements.
